Welcome to Sec-Aware, a small web application demo created for my project. The questions on the various games are based around various topics surrounding human error in cyber security and what they can lead to.


The web application has no specific order, but it is recommended to complete it using the following steps:

1. If needed, visit the Information page to practice for the game questions.

2. Visit the "select a game" page.

3. Play the Beginner game.

4. Play the Intermediate game.

5. Play the Expert game.

6. Visit the leaderboards.

7. Send feedback via the feedback page.
